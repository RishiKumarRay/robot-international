Overview

Wholesomemaker using Flask as Website Framework.

  • app.py Flask Framework in Python, obviously.
  • freezer.py Static Website converter (Flask Freezer)
  • /templates Website templates in HTML (Flask formatted)
  • /static web assets (css, js, and image)

Frontend and Backend Development

To work on the frontend, you mostly only need to focus on the /templates directory.

Run python app.py to start working on frontend development.

The frontend is deployed directly to Github Pages from the gh-pages branch using this Github Actions.

On the other hand, Website backend are the app.py itself.
